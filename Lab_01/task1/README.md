# Lab 1.1 – Imperative vs Declarative

## Императивная реализация
Реализация выполнена с использованием jQuery.
При нажатии на кнопку происходит добавление или удаление CSS-класса у элемента
путём прямого изменения DOM.

## Декларативная реализация
Реализация выполнена с использованием React.
Состояние интерфейса хранится в переменной состояния, а React автоматически
обновляет отображение при её изменении.

## Запуск React-версии
npm install
npm run dev
