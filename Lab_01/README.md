# Лабораторная работа №1 – React Fundamentals

Данная работа выполнена в рамках курса по React (Week 1).

## Структура проекта
- task1 – реализация одного и того же поведения двумя способами:
  - императивный подход (jQuery)
  - декларативный подход (React)
- task2 – React-приложение со счётчиком

## Task 1
В первом задании реализовано переключение подсветки текста по нажатию на кнопку:
- в императивной версии используется jQuery и прямое управление DOM
- в декларативной версии используется React и состояние (useState)

## Task 2
Во втором задании реализован простой счётчик на React.
Значение счётчика можно увеличивать и уменьшать с помощью кнопок.

## Используемые технологии
- Node.js
- Vite
- React 18

## Как запустить React-проекты
1. Перейти в папку проекта
2. Выполнить команду `npm install`
3. Выполнить команду `npm run dev`
