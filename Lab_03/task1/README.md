# Lab 3.1 – Step Counter

В данной лабораторной работе реализован компонент StepCounter,
который использует props для передачи начальных значений
и useState для управления состоянием компонента.

Компонент поддерживает:
- изменение значения с заданным шагом
- хранение истории изменений
- подсчёт количества операций
- сброс состояния
- работу нескольких независимых счётчиков

## Запуск проекта
npm install
npm run dev
