# Lab 3.2 – User Profile

В данной лабораторной работе реализован компонент UserProfile,
который загружает данные пользователя с использованием useEffect.

Реализованы:
- загрузка данных через fetch
- состояния loading и error
- AbortController для очистки запроса
- повторная загрузка данных при изменении userId

## API
https://jsonplaceholder.typicode.com/users

## Запуск проекта
npm install
npm run dev
